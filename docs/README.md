# suisoku docs

Vercel 向けの最小構成を配置しています。

ハナビの画面では、入力欄の名前を左、入力欄を右に並べています。
推測ボタンを押すと、入力値から実践G数や各確率を計算して結果を表示します。BIG中の項目は、BIG中の消化G数を分母に使います。
さらに、BIG、REG、通常時小役、BIG中、花火チャレンジ中、花火ゲーム中について、各項目ごとの設定別割合を表示します。
その8要素をまとめた、設定1・2・5・6の総合推測も表示します。
各項目ごとの設定別割合も、下のスペック表と同じまとまりごとの表で整理しています。
設定表は、基本スペック、通常時小役、BIG中、花火チャレンジ中、花火ゲーム中のまとまりごとに分けています。
全体の見た目は、スマホで見やすいように横幅と余白を詰めています。
スマホでも、3項目の行は横3列のまま、ラベルを入力欄の左隣に置く形にしています。
